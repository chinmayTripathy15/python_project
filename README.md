# 🧮 Smart CLI Calculator

A simple yet powerful **command-line calculator** built using Python.  
It supports real-world mathematical expressions, history tracking, and file storage.

---

## 🚀 Features

- ✅ **Natural Expression Input**
  - Supports expressions like:
    ```
    7 + (7 - 7) * 1
    10 / 2 + 5
    ```

- 🧠 **Multi-Operation Support**
  - Handles `+`, `-`, `*`, `/`
  - Supports brackets `()` and operator precedence

- 🕒 **Timestamped History**
  - Every calculation is saved with **date & time**

- 📂 **Persistent Storage**
  - History is saved in `history.txt`

- 📜 **History View**
  - Press `h` to view past calculations

- 🧹 **Clear History**
  - Press `cl` to clear all history

- ⚠️ **Error Handling**
  - Handles invalid inputs and division by zero

---

## 🖥️ How to Run

### 1. Go to project folder
```bash
cd smart-calculator
```


###  2. Make sure Python is installed
```bash
python --version
```

### 3. Run the program
``` bash
python calculator.py
```
